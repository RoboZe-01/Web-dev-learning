What is JSON ?

It is similar like js object but key is enclosed in double quotes

  • Common in networks calls and data storage (Works across all programming languages)
  • JSON.stringify and JSON.parse()
  • Json.stringify
    Convrts js objects to json string
  • JSON.parse()
    COnverts json string into js object
  • Strings are very easy to transport over a network
  • JSON is a data format and js object is a data structure
