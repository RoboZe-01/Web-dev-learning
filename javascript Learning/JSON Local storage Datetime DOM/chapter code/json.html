What is JSON ?

It is similar like js object but key is enclosed in double quotes

  • Common in networks calls and data storage (Works across all programming languages)
  • JSON.stringify and JSON.parse()
  • Json.stringify
    Convrts js objects to json string
  • JSON.parse()
    COnverts json string into js object
  • Strings are very easy to transport over a network
  • JSON is a data format and js object is a data structure

Local starage

Things to know

  1. Persistent, key-value string storage in the browser.
  2. Requires JSON.stringify() for objects/arrays and JSON.parse() for retrieval..
  3. Data is isolated per website origin (protocol, domain, port).
  4. Has a limited storage capacity (around 5-10 MB).
  5. Do not store any sensitive information here as it is publically viewables

Methods in Local storage

  • setItem(key, value): Adds a key-value pair to local storage or updates the value if the key already exists.
  • getItem(key): Retrieves the value associated with a specific key from local storage, returning null if the key doesn't exist.
  • clear(): Empties all key-value pairs currently stored in the local storage for the origin.
  • removeItem(key): Removes the key-value pair associated with a given key from local storage
  • key(index): Returns the name (key) of the key at the specified zero-based index in the local storage.
  • length: Returns the number of key-value pairs currently stored in the local storage.
